  • mysql 数据库检测

    # mysql 数据库检测

    show variables like'%max_connection%'; 查看最大连接数

    show status like 'Threads%'; Threads_connected ：这个数值指的是打开的连接数. Threads_running ：这个数值指的是激活的连接数，这个数值一般远低于connected数值. Threads_connected 跟show processlist结果相同，表示当前连接数。准确的来说，Threads_running是代表当前并发数

    show full processlist; 显示当前所有连接信心

    select * from information_schema.innodb_trx; 查询正在执行的事务:INNODB_TRX

    • kill 线程id 杀掉线程

    SELECT * FROM INFORMATION_SCHEMA.INNODB_LOCKS; 查询正在锁的事务 SELECT * FROM INFORMATION_SCHEMA.INNODB_LOCK_WAITS; 查询等待锁的事务: select * from information_schema.PROCESSLIST where info is not null; 查询执行中的进程

    # 查询mysql数据库中是否存在某个字段

    select table_name from information_schema.columns where table_schema = '数据库名' and column_name='字段名';
